Custom Programming Language Project

MineScript

A Minecraft-inspired programming language that turns code into a text-based RPG adventure.

See Examples View GitHub Repo

What is MineScript?

MineScript is a Minecraft-themed programming language designed to make programming more fun, visual, and beginner-friendly. Instead of writing programs with only traditional keywords, users write code as actions performed by Steve, mobs, and player objects.

Programs are written in .ms files and executed by a Python interpreter called minescript.py. The output is text-based narration in the terminal, making the program feel like a small RPG adventure.

Project Purpose

The goal of MineScript is to help beginners understand core programming concepts through a familiar game-inspired framework. Concepts like variables, conditionals, loops, objects, functions, and output are mapped to Minecraft-style commands.

Language Features

Object-Oriented Design

Steve is the main object. Mobs and optional players can also be created with their own attributes.

Python Interpreter

minescript.py reads .ms files and executes each line as MineScript code.

Terminal Narration

Programs produce story-like output in the terminal using commands such as /chat and /attack.

RPG Syntax

Programming structures are renamed with game terms like encounter, mine, and loot.

Core Syntax

Main Player

Steve(HP, Attack, Armor)

Mob Object

Creeper(HP, Attack)

Integer Variable

Steve.carry(identifier, value)

String Variable

Steve.nametag(identifier, "value")

If / Else

Steve.encounter(condition) {
  /chat "true path"
} otherwise {
  /chat "false path"
}

While Loop

Steve.mine(condition) {
  /chat "looping"
}

For Loop

Steve.loot(i = 1, i <= 10, i++) {
  /chat i
}

Output

/chat "Hello world"
/chat Steve.hp

How to Run a MineScript Program

The interpreter is included in the GitHub repository. To run a program, use Python 3 and pass the .ms file as an argument. 

python3 minescript.py combatSimulator.ms
python3 minescript.py greetingProgram.ms
python3 minescript.py fizzBuzz.ms

Storyboard / Image Generation Extension

MineScript also explores an image-based storytelling extension. The idea is that the terminal narration from a MineScript program can be converted into visual storyboard panels, making the code feel even more like a Minecraft adventure.

For example, a combat program can become a panel showing Steve fighting a Creeper, while a FizzBuzz program can become a visual number-block puzzle.
