About the Language

MineScript is a simple custom programming language inspired by Minecraft. The language uses game-like commands to help make programming concepts easier to understand. The interpreter is written in Python and runs MineScript programs stored in .ms files.

Greeting Program

This program demonstrates basic output in MineScript. 

# greetingProgram.ms

say "Welcome to MineScript!"
say "A Minecraft-inspired programming language."

FizzBuzz Program

This program demonstrates conditional logic and repeated output. 

# fizzBuzz.ms

repeat 1 to 20
  if number divisible by 3 and 5
    say "FizzBuzz"
  else if number divisible by 3
    say "Fizz"
  else if number divisible by 5
    say "Buzz"
  else
    say number

Combat Simulator

This program shows how MineScript can represent a simple game-like battle. 

# combatSimulator.ms

spawn player
spawn zombie

player attack zombie
if zombie health less than 1
  say "Zombie defeated!"
else
  zombie attack player

Story Program

This program shows how MineScript can create a short story sequence. 

# story.ms

say "The player wakes up in a dark cave."
move forward
mine stone
craft torch
say "The cave lights up."

Running the Interpreter

The interpreter is included in the GitHub repository as minescript.py. Example commands: 

python3 minescript.py greetingProgram.ms
python3 minescript.py fizzBuzz.ms
python3 minescript.py combatSimulator.ms
python3 minescript.py story.ms
