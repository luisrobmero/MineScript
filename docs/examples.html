Language Overview

MineScript is a Minecraft-style RPG programming language designed to make programming more engaging for beginners. The language is object-oriented, interpreted, and runs sequentially. Steve is the main player object, while mobs and optional player characters can also be created.

The language replaces common programming structures with game-inspired syntax. For example, Steve.encounter(condition) works like an if statement, Steve.mine(condition) works like a while loop, and Steve.loot(init, condition, increment) works like a for loop. Slash commands such as /chat, /attack, /sleep, and /eat act like built-in functions.

Core Syntax

Main Player Object

Steve(HP, Attack, Armor)

Mob Object

Creeper(HP, Attack)

If / Else

Steve.encounter(condition) {
  /chat "true path"
} otherwise {
  /chat "false path"
}

While Loop

Steve.mine(condition) {
  /chat "looping"
}

For Loop

Steve.loot(i = 1, i <= 100, i++) {
  /chat i
}

Output

/chat "Hello world"
/chat Steve.hp

Example 1: Combat Simulator

This example creates Steve and a Creeper, then uses a while loop to keep attacking until the Creeper's HP reaches zero. It demonstrates objects, loops, conditionals, and slash commands. 

#define player and mob objects
Steve(20, 10, 5)
Creeper(15, 8)

#While loop: Fights until Creeper HP reaches 0
Steve.mine(Creeper.hp > 0) {
  /attack Steve Creeper
  /attack Creeper Steve
  /chat Creeper.hp
  /chat Steve.hp
}

#Determine the winner and create output
Steve.encounter(Creeper.hp <= 0) {
  /chat "Steve wins!"
} otherwise {
  /chat "Creeper wins!"
}

Expected Terminal Output

Steve mines his way to a Creeper
Creeper's HP is reduced to 0
Steve wins!

Example 2: Greeting Program

This example defines Steve and two optional player characters. The higher-level player greets the lower-level player using a user-defined slash command. 

#Define Player objects
Steve(20, 10, 5)
Alex("Alex", 5)
Jordan("Jordan", 10)

#Define Greet Function
/greet greeter receiver {
  /chat greeter.name
  /chat "greets"
  /chat receiver.name
  /chat "Welcome, traveler!"
}

#Higher level player will greet other players
Steve.encounter(Alex.level > Jordan.level) {
  /greet Alex Jordan
} otherwise {
  /greet Jordan Alex
}

Expected Terminal Output

Jordan greets Alex
Welcome, traveler!

Example 3: FizzBuzz

This example shows a classic FizzBuzz program using MineScript's for-loop structure, nested conditionals, modulo operations, and terminal output. 

#Start the program
Steve(20, 10, 5)

#Iterate through the desired interval
Steve.loot(i = 1, i <= 100, i++) {
  Steve.encounter((i % 15) == 0) {
    /chat "FizzBuzz"
  } otherwise {
    Steve.encounter((i % 3) == 0) {
      /chat "Fizz"
    } otherwise {
      Steve.encounter((i % 5) == 0) {
        /chat "Buzz"
      } otherwise {
        /chat i
      }
    }
  }
}

Running the Interpreter

The MineScript interpreter is written in Python and included in the GitHub repository as minescript.py. Example programs are stored as .ms files. 

python3 minescript.py combatSimulator.ms
python3 minescript.py greetingProgram.ms
python3 minescript.py fizzBuzz.ms
